Auberon

The classic game of Connect-4 against an AI player. New in this release:

@DESCRIPTION@

Download Auberon @VERSION@ (0.25 MB disk image). This is a Universal Binary, and should run on Mac OS X 10.3.9 and later.

Multiple difficulty levels are supported. In creating this game I borrowed some high-quality graphics from Gnome Iagno.

Versions of this game prior to version 0.3 were called "Puck".

Auberon is written and maintained by Stig Brautaset. This program is Free Software, and released under the GPL. You can get the source from my Subversion repository. If you want to hack on Auberon you also need SBAlphaBeta.
